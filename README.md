# 🌌 ELITE 99/10: Unified AI Multi-Agent SaaS

This is an enterprise-grade AI solution delivered as a clean, single-process **Flask SaaS**. It eliminates modern complexity while maintaining high-end visual standards (Apple-style Glassmorphism).

## 🚀 The Intelligence Squad
1. **🔍 Research Agent:** Real-time news extraction via Perplexity & Global web indexing.
2. **📊 Analysis Agent:** Strategic deep-dive, market tiering, and ROI prediction using GPT-4o.
3. **✉️ Outreach Agent:** Generates "Un-ignorable" copy for LinkedIn/Email.

## 🌈 Professional UI
A single-terminal dashboard built with **Tailwind CSS** and **Framer Motion**, featuring:
- **Lead Probability Metric:** Instant visual feedback on lead quality.
- **Mission Log:** Real-time terminal-style updates showing agent progress.
- **Strategic Matrix:** Categorized views for gaps, news, and copy.

## 🛠️ Setup & Execution
1. **Environment:**
   ```env
   OPENAI_API_KEY=...
   PERPLEXITY_API_KEY=...
   SERPER_API_KEY=...
   ```
2. **Install:** `pip install -r requirements.txt`
3. **Launch:** `python app.py`
4. **Access:** Open `http://localhost:5000`
